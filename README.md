# 🏛️ نظام حزب المستقبل لإدارة البيانات الانتخابية والتنظيمية

نظام ويب متكامل واحترافي مصمم خصيصاً لإدارة بيانات حزب سياسي وهياكله التنظيمية (الأحياء والشوارع والأعضاء). يتميز بمظهر عصري وجذاب ولوحة تحكم متكاملة تتناسب تماماً مع الهواتف والحواسيب.

---

## 🛠️ التقنيات المستخدمة في المشروع

* **الواجهة الأمامية:** HTML5, CSS3, JavaScript.
* **إطار التصميم:** Bootstrap 5 (RTL النسخة المتوافقة مع اللغة العربية) مع أيقونات Bootstrap Icons.
* **الواجهة الخلفية:** PHP 8 (مع التوافق الكامل مع إصدارات PHP 7.4+).
* **قاعدة البيانات:** MySQL عبر مكتبة PDO وقواعد البيانات العلائقية (Relational Database).
* **التصميم والجماليات:** لوحة ألوان هادئة ومريحة تعتمد على درجات الأخضر الداكن والأبيض والرمادي، خط Cairo للواجهات العربية، ميزات شبه شفافة (Glassmorphism)، ورسوم بيانية تفاعلية باستخدام Chart.js.

---

## 🔒 الميزات الأمنية المطبقة

1. **حماية ضد SQL Injection:** استخدام الاستعلامات المحضرة (Prepared Statements) والمعاملات المربوطة بشكل كامل عبر PDO في جميع استعلامات قاعدة البيانات.
2. **تشفير كلمات المرور:** تشفير كلمات المرور باستخدام خوارزمية التجزئة الحديثة `password_hash()` والتحقق عبر `password_verify()`.
3. **حماية ضد هجمات CSRF:** توليد وتدقيق رموز التحقق الأمنية التلقائية في جميع النماذج التي تقوم بعمليات التعديل أو الحذف أو الإضافة.
4. **حماية ضد هجمات XSS:** تنظيف وعزل كافة المدخلات المعروضة في صفحات المتصفح باستخدام مصفوفة تنظيف متقدمة (`htmlspecialchars`).
5. **جدار عزل الصلاحيات:** لا يمكن لمشرف أي حي الوصول بأي شكل من الأشكال (حتى عبر التلاعب بالروابط وعناوين URL) إلى بيانات حي آخر أو شوارع أو أشخاص لا يخضعون لإشرافه المباشر.

## 🚀 طريقة التثبيت والتشغيل (بالتفصيل)

### 1. المتطلبات الأساسية
تأكد من تثبيت بيئة عمل خادم محلي على جهازك، مثل:
* **XAMPP** (أو WAMP Server أو Laragon) والذي يحتوي على خادم PHP وخادم MySQL.

### 2. خطوات نقل المشروع
1. قم بإنشاء مجلد جديد في مسار مشاريع الخادم المحلي الخاص بك (htdocs في XAMPP):
   * المسار الافتراضي: `C:\xampp\htdocs\political_party`
2. انقل جميع ملفات ومجلدات المشروع إلى هذا المجلد.

### 3. تشغيل معالج التثبيت التلقائي (Web Installer)
يحتوي النظام على معالج تثبيت رسومي ذكي ومبسط يغنيك عن استيراد ملفات SQL أو تعديل الأكواد يدويًا:
1. تأكد من تشغيل كل من **Apache** و **MySQL** في لوحة تحكم XAMPP.
2. افتح متصفح الويب الخاص بك وانتقل إلى الرابط التالي:
   * **`http://localhost/political_party/install.php`**
3. سيظهر لك معالج التثبيت باللون الأخضر:
   * **إعدادات قاعدة البيانات:** أدخل عنوان الخادم (`localhost`)، واسم قاعدة البيانات التي ترغب بها (مثلاً: `political_party_db`)، واسم المستخدم لـ MySQL (الافتراضي هو `root` في XAMPP)، وكلمة المرور (اتركها فارغة في XAMPP).
   * **إعداد حساب المدير العام:** أدخل اسم المستخدم للمدير العام (مثلاً `admin`) وكلمة المرور الخاصة به (مثلاً `admin123`).
4. اضغط على زر **"تهيئة قاعدة البيانات وتثبيت النظام"**.
5. سيقوم المعالج تلقائياً بإنشاء قاعدة البيانات، وبناء الجداول والروابط، وتوليد 4 أحياء افتراضية وبداخل كل حي 29 زنقة (زنقة 1 إلى زنقة 29)، ثم تشفير حساب المدير وحفظ ملف التكوين بنجاح. بعد ذلك، سيتم توجيهك مباشرة لصفحة تسجيل الدخول.

## 👥 نظام الصلاحيات ومستخدمي النظام الافتراضيين

بعد تشغيل معالج التثبيت، يمكنك تسجيل الدخول باستخدام الحسابين التاليين للتجربة:

### 1. حساب المدير العام (Admin) - كامل الصلاحيات:
* **اسم المستخدم:** `admin` (أو الاسم الذي اخترته أثناء التثبيت)
* **كلمة المرور:** كلمة المرور التي اخترتها أثناء التثبيت (مثال: `admin123`)
* **الصلاحيات:**
  * رؤية إحصائيات عامة ورسم بياني لجميع الأحياء.
  * استعراض ودخول جميع الأحياء الـ 4، والشوارع الـ 29 لكل حي.
  * إدارة الأعضاء والمواطنين (إضافة، تعديل، حذف، وبحث وفرز).
  * تصدير البيانات إلى Excel وطباعة قوائم الـ PDF.
  * إدارة مستخدمي النظام وإنشاء مسؤولين جدد وتعيين الأحياء الخاصة بهم.
  * الاطلاع على سجل العمليات والأنشطة بالكامل لمعرفة من قام بأي تعديل بالنظام والتوقيت وعنوان الـ IP.
  * أخذ نسخ احتياطية كاملة من قاعدة البيانات وتنزيلها كملف `.sql` أو استعادة نسخ سابقة.

### 2. حساب مسؤول الحي (Manager) - صلاحيات محددة:
لإنشاء حساب مسؤول حي، قم بالدخول بحساب المدير العام ثم انتقل لـ **"إدارة المستخدمين"** وأنشئ حسابًا جديدًا مرتبطًا بالحي الأول "الحي المحمدي" مثلاً (اسم المستخدم: `manager_riad` وكلمة المرور: `manager123`).
عند تسجيل الدخول بهذا الحساب:
* **الصلاحيات:**
  * يرى إحصائيات الحي المحمدي فقط.
  * يرى فقط الحي المحمدي في لوحة التحكم، وباقي الأحياء يتم حجبها تمامًا.
  * يمكنه إدارة الشوارع الـ 29 الخاصة بالحي المحمدي وإدارة مواطنيها بالكامل.
  * يرى فقط سجل العمليات الخاص بالأنشطة التي قام بها هو شخصياً.
  * يُحجب تمامًا عن صفحات الإشراف العام (إدارة المستخدمين، النسخ الاحتياطي، سجل العمليات الكلي) ويتم تحويله تلقائياً للرئيسية في حال محاولة تخطي ذلك.

## 📁 هيكلية ملفات المشروع للتطوير

* **`config/database.php`:** يحتوي على اتصال قاعدة البيانات عبر PDO مع توجيه ذكي للمعالج في حال عدم اكتمال التثبيت.
* **`config/auth.php`:** عقل الحماية بالخلفية، يتأكد من الجلسات، التحقق من الصلاحيات ورموز الـ CSRF، وسجل العمليات.
* **`includes/header.php` & `includes/footer.php`:** القوالب البصرية المشتركة لضمان ثبات المظهر وسرعة التحميل.
* **`assets/css/style.css`:** المظهر الجمالي المخصص وألوان الهوية التفاعلية وتهيئة التجاوب وعزل عناصر الطباعة.
* **`assets/js/app.js`:** التحكم التفاعلي بالطرف العميل (تجاوب القائمة، رسائل التأكيد قبل الحذف، التحقق من مطابقة كلمات المرور وتنسيق بطاقات التعريف الوطنية).
* **`people.php` & `person-actions.php`:** واجهة وخلفية إدارة الأشخاص والمواطنين (CRUD) والبحث متعدد المعايير.
* **`export.php`:** أداة تحويل البيانات وتنزيلها بصيغة Excel (CSV) متوافقة مع اللغة العربية.
* **`backup.php`:** لوحة النسخ الاحتياطي الذكي والاستعادة المباشرة.
